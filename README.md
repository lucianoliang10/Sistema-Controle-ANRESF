# Sistema Controle ANRESF

Sistema web para acompanhamento de processos e casos da Unidade Técnica da ANRESF.

## Objetivo

Criar uma primeira versão estática e demonstrativa para organizar a visualização de casos, prazos, fluxos processuais, sanções e controle de numeração documental.

## Stack inicial

- HTML
- CSS
- JavaScript puro

## Status

MVP estático inicial, sem conexão com banco de dados, sem APIs e sem autenticação. Todos os dados exibidos são fictícios e servem apenas para demonstração.

## Como executar

Abra o arquivo `index.html` diretamente no navegador ou publique os arquivos como um site estático na Vercel.
